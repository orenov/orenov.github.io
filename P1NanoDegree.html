Section 1. Statistical Test¶

The first question which i asked myself was: Is there any significant difference between number of people enetering subway in rainy and non rainy days. As shown below, data for this particular question isn't normally distributed. So it's suitable to use non-parametric test. In class i've learnt Mann–Whitney U test. Null hypothesis for our two-sided test will be is P(x>y)=0.5, where x - subway crowd in rainy days, y - subway crowd in non rainy days. At first let's check our assumptions about non-gaussian data using Shapiro-Wilk test:

In [5]: 
scipy.stats.shapiro(rain)
Out[5]:
(0.4715914726257324, 0.0)
In [6]: 
scipy.stats.shapiro(non_rain)
Out[6]:
(0.47661787271499634, 0.0)

As we can see from Shapiro-Wilk test our data is likely non normal distributed due to extremely low p-value. No let's calculate statistics and p-value for Mann Whitney U test:

In [7]: 
t, p_value = scipy.stats.mannwhitneyu(rain, non_rain)
2 * p_value
Out[7]:
0.049999825586979442

We've multiplied p-value by two, because scipy returns p-value for one-sided test. This value is less than 0.05 - our significant level, so we should reject the null hypothesis about same mean Entries in rainy or not days. Let examine this data more and look at summary.

Section 4. Conclusion¶

In this first project we partially investigate NYC Subway Dataset. I was trying to answer next question: "Do more people ride the NYC subway when it is raining or when it is not raining?". As part of this investigation i've obtain two samples (rainy and non rainy days). I needed to check zero hypothesis that means of two samples are equal. At first i need to choose statistical test to do this task correctly. I've checked both samples on normality test. Shapiro-Wilk test rejected zero hypothesis about data normality for both samples. So i've choosen Mann-Whitney U test or Wilcoxon rank test, becuase it is non-parametric test (no assumptions about distributions). For our tow samples Mann-Whitney U test has rejected hypothesis. So we can conclude that distribution of the number of entries statistically different between rainy and non rainy days with signifficant level 5%. Let's look at this question from Linear Regression side. As we can see from ols summary, rain coefficient include zero in 95% confidence interval, so from this perespective we can conclude that rain doesn't make any difference in number of people riding the NYC subway. There is still need more investigation to make right decision, but anyway i believe in non-parametric approach more. I've build ols model with not so good r-square equals to 0.557. But it's can be initial solution before more deeper diving in this dataset.

At the end of conclusion i want to citate Mark Twain: "Lies, damned lies, and statistics". Who knows? OLS or Mann-Whitney U test?
